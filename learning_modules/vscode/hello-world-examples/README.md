# Using Generic Dev Container Examples

## C++

```cpp
#include <iostream>

int main()
{
    std::cout << "Hello World!\n";
    return 0;
}
```

Compile with: `g++ main.cpp`
Run with: `./a.out`

## Python

```py
print("Hello World!\n")
```

Run with: `python3 main.py`

## Golang

```go
package main

import "fmt"

func main() 
{
    fmt.Println("Hello World!")
}
```

Build with: `go build main.go`
Run with: `./main`
